# HOME.IAI.ONE

HOME.IAI.ONE is the central system portal of the wider IAI structure.

It does not replace the Charter, does not operate ecosystems, and does not act as the technology flagship itself. Its role is to help people see the system clearly before entering any specific layer.

## Core Definition

- `iai.one` is the Charter layer
- `home.iai.one` is the Portal layer
- `flow.iai.one` is the Infrastructure flagship

## Purpose of Home

HOME exists to make the wider IAI system understandable through:

- system definition
- role clarity
- ecosystem mapping
- correct entry paths
- visible boundaries
